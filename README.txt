# IceCubev1 - IceCube in SNOwGLoBES (v1)



./snowglobes/ contains the smearing files and post-smearing efficiencies used to run IceCubev1.

./snowglobes/doc contains a design document explaining how these experiment files were generated.

## Detector configurations

The file ./snowglobes/detector_configurations.dat contains those detector configurations present in SNOwGLoBES v1.2

If you wish to add IceCubev1 to your detector_configurations.dat file without replacing it, simply add the line

IceCubev1      51600.   0.1111
